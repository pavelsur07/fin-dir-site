# Tokens (цвета / типографика / базовые принципы)

## Цвета бренда (зафиксировано)
- Brand (primary): `#0A1548`
- Brand (secondary): `#0d1c63`
- Accent (CTA / red): `#B00020` (в UI может встречаться как более глубокий красный)
- Muted text: используется переменная темы `--vf-muted` (не хардкодить без необходимости)

## Типографика (принцип)
- Заголовки: жирные, плотные, B2B-стиль
- Подзаголовки/лиды: читабельные, спокойные, без маркетинговых «восклицаний»
- Межстрочные:
    - H1: ~1.06
    - lead: ~1.55–1.6

## Принципы визуального языка
- Минимализм, аккуратные карточки
- Мягкие тени (не «пластик»)
- Скругления: 6px–16px в зависимости от компонента
- CTA: всегда понятный primary + secondary (outline)

## Запрещено
- Ядовитые градиенты, “инфобиз”-перегрев
- Временные цвета/токены без фиксации в документации
