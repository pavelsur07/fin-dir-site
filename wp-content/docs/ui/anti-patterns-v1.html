UI System Anti‑паттерны (Blue Ocean, Enterprise B2B)

Anti‑паттерны: как не стать “default SaaS”

Это демонстрационный шаблон. Он фиксирует “запреты” и эталонные решения для дальнейшей верстки: меньше шума, больше управляемости и доверия.

1 filled CTA на секцию Без градиентов/стекла 56–64ch для текста Радиусы 4–8px

Цвет: нельзя

❌ Anti‑pattern
Много акцентов и “синий по умолчанию”
2–3 CTA‑цвета в одном блоке, “вкусные” градиенты, промо‑красный.
Выглядит как типовой SaaS‑лендинг.

Цвет: как надо

✅ Pattern
Один акцент и роли цветов
--brand для структуры, --cta-2 (брендовый) для “начать”, --cta для “ответственного” шага.
Сдержанно, управляемо, enterprise‑тон.

Типографика: нельзя

❌ Anti‑pattern
Очень эмоциональный заголовок!!!
Подзаголовок в стиле промо
Длинный текст на всю ширину экрана — читать тяжело, выглядит как “лендинг‑простыня”.
Слишком много ролей текста и “эффектов”.

Типографика: как надо

✅ Pattern
Контроль денег и прогноз кассы
Текст ограничен до 62ch. 3 роли текста: управляющий / основной / служебный.
Спокойно, понятно, “рабочая тишина”.

Чек‑лист перед мерджем

  • В секции не больше 1 filled CTA
  • Нет градиентов / glass / неонов
  • Текст ограничен по ширине (56–64ch)
  • Радиусы в пределах 4–8px
  • Цвет = роль (структура/действие/риск), а не “красота”
