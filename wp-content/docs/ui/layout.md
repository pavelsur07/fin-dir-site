# Layout (каркас страниц)

Документ фиксирует базовые принципы каркаса страниц темы.

## Принцип
- Страница собирается из template-parts блоков
- Изменения в общих блоках должны быть безопасны для всех страниц

## Hero
- Главная: `hero-front-page.php` (Hero4)
- Service pages: `hero-wide.php` (Hero3)

## Изоляция
- Стили блоков должны быть scoped:
    - `.vf-hero4 ...`
    - `.vf-hero3 ...`
