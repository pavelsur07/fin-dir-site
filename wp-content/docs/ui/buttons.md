# Buttons (CTA) — эталон

## Назначение
Кнопки — базовый компонент темы. В проекте используются два основных CTA-стиля:
- Primary CTA (filled)
- Secondary CTA (outline)

---

## Используемые классы
### Primary CTA
- `.btn.btn-cta`

### Secondary CTA
- `.btn.btn-cta-secondary`

---

## Базовые требования
- Высота: `min-height: 44px`
- Padding: `10px 14px`
- Radius: `6px`
- Font-weight: `900`
- Line-height: `1`

---

## Mobile правило для hero
В hero-блоках допускается (и используется):
- CTA в колонку
- каждая кнопка `width: 100%`

---

## Запрещено
- уменьшать CTA ниже 44px по высоте
- добавлять emoji в CTA-текст
- вводить новые варианты кнопок без фиксации в документации
