# Cards (карточки)

Карточки — основной визуальный контейнер в теме.

## Принцип
- Card = фон + border + мягкая тень + аккуратный radius
- Card не должна выглядеть как "пластик" или "инфобиз"

## Базовые параметры (ориентир)
- border: `rgba(10,21,72,0.10)`
- radius: 6px
- shadow: мягкая, B2B

## Запрещено
- glass/blur эффекты
- неоновые тени
- агрессивные градиенты
