# STRUCTURE

- `style.css`: WordPress theme header.
- `functions.php`: Loads theme setup, enqueues, helpers.
- `assets/`: CSS, JS, and images.
- `inc/`: Setup, enqueue, helper functions.
- `template-parts/`: Shared layout parts such as header and footer.
- `templates/`: Page templates for specific layouts.
- `docs/`: Theme guidelines and release notes.
